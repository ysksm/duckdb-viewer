# Requirements - DuckDB Viewer

## 概要

DuckDB データベースのデータを閲覧・操作するためのデスクトップアプリケーション。
Tauri + Angular v21 で構築する。

## 機能要件

### 1. データベース接続

- [ ] DuckDB ファイル（.duckdb, .db）の選択・オープン
- [ ] 新規データベースの作成
- [ ] インメモリデータベースの作成
- [ ] 接続履歴の保存・表示

### 2. テーブル閲覧

- [ ] データベース内のテーブル一覧表示
- [ ] テーブルのスキーマ情報表示（カラム名、型、制約）
- [ ] テーブルデータのページネーション表示
- [ ] カラムによるソート機能
- [ ] フィルタリング機能

### 3. SQL 実行

- [ ] SQL エディタ（シンタックスハイライト付き）
- [ ] SQL の実行と結果表示
- [ ] 複数クエリの実行対応
- [ ] クエリ履歴の保存
- [ ] クエリの保存・読み込み
- [ ] 実行時間の表示
- [ ] エラーメッセージの表示

### 4. データエクスポート

- [ ] CSV 形式でのエクスポート
- [ ] Excel（.xlsx）形式でのエクスポート
- [ ] Parquet 形式でのエクスポート
- [ ] JSON 形式でのエクスポート
- [ ] エクスポート対象の選択（全データ / 選択行 / クエリ結果）

### 5. データインポート

- [ ] CSV ファイルのインポート
- [ ] Parquet ファイルのインポート
- [ ] Excel ファイルのインポート

### 6. ダッシュボード

- [ ] ダッシュボードの作成・編集
- [ ] チャート/グラフウィジェット（棒グラフ、折れ線グラフ、円グラフ）
- [ ] テーブルウィジェット
- [ ] KPI/メトリクスウィジェット
- [ ] ウィジェットのドラッグ&ドロップ配置
- [ ] ダッシュボード設定の永続化（ローカルストレージ/ファイル）
- [ ] 複数ダッシュボードの管理
- [ ] データソース（クエリ）とウィジェットの紐付け

## 非機能要件

### パフォーマンス

- 100万行以上のデータでも快適に動作
- 大量データのページネーション表示
- 非同期処理による UI のブロック防止

### ユーザビリティ

- 直感的な UI/UX
- キーボードショートカット対応
- ダークモード対応

### セキュリティ

- ローカルファイルへのアクセスのみ
- 外部ネットワーク通信なし

## 技術スタック

### フロントエンド

- Angular 21
- TypeScript
- Angular Material（UI コンポーネント）
- Monaco Editor（SQL エディタ）

### バックエンド

- Tauri 2.x（Rust）
- DuckDB Rust SDK

### ビルド・開発

- Node.js 20+
- Rust 1.70+
- Cargo
